# UIHEventAdder
EventAdder is an Outlook VSTO plugin that automatically converts incoming emails and meeting invitations into calendar appointments—complete with attachments—and forwards them to a configurable recipient while preventing infinite loops.
# 下载app.publish.zip压缩包，双击.vsto文件安装
** 1.关闭outlook 2.双击.vsto文件安装 3.启动outlook，在加载项页面输入自己的邮箱，保存设置 4.使用保存键下方的测试键可测试是否成功，成功后可在Feishu日历收到转发的新邮件提醒（无回复功能）。
